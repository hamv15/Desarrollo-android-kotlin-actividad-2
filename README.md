
# Desarrollo Android - Kotlin: Actividad 2

Proyecto dónde se genera una interfaz gráfica con grupos de vistas, tales como:

- ✅ FrameLayout
- ✅ LinearLayout
- ✅ RelativeLayout
- ✅ ConstraintLayout
- ✅ RecyclerLayout

Adicionalmente, se utilizan los siguientes elementos:

- ✅ TextView
- ✅ ImageView
- ✅ EditText
- ✅ Button
- ✅ Checkbox
- ✅ RadioButton
- ✅ Spinner
